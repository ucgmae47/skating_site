# Athens Figure Skating Club Website

A full-stack web application built to provide an interactive platform for members of the Athens Figure Skating Club. The project demonstrates secure user authentication, database management, front-end and back-end integration, and full-stack web development principles.

---

## Project Overview

This web application allows club members to:

- View upcoming events and schedules  
- Access and update member profiles  
- Securely log in with role-based access controls  
- Administer content and manage events  

The project showcases a combination of front-end and back-end skills, including template rendering, responsive design, and database integration.

---

## Key Features

- **Full-stack development** with Flask (Python), MySQL, HTML, CSS, and JavaScript  
- **Secure authentication** with password hashing and role-based access control  
- **ORM with SQLAlchemy** for efficient database interactions, including schema management and migrations  
- **App routing and template rendering** for modular, maintainable code  
- **Integration of front-end and back-end** to provide a seamless user experience  

---

## Technologies

- **Backend:** Python, Flask, SQLAlchemy  
- **Database:** MySQL  
- **Frontend:** HTML5, CSS3, JavaScript  
- **Authentication & Security:** Password hashing, role-based access
